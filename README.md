# My Art Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.  
Live at: https://lumiarts.github.io

## About
A single-page portfolio showcasing my digital illustrations and paintings,
with a gallery, about section, and contact form.

## Structure
- `index.html` — the entire site (HTML, CSS, and JS in one file)
- `images/` —

- ## Customizing
- Open `index.html` in any text editor
- Update the `works` array (~line 290) to add your real artwork
- Replace placeholder bio, stats, and links with your own info
- For the contact form, sign up at formspree.io and add your form ID

## Contact Form
Uses [Formspree](https://formspree.io) for form submissions (free tier available).
